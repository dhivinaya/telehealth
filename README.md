# 🩺 Online Doctor Health Monitoring System

A Flask-based web app for real-time doctor-patient interaction, appointment booking, prescriptions, and health monitoring.

## 👨‍💻 Tech Used
- Frontend: HTML, CSS, JavaScript
- Backend: Python Flask
- Database: SQLite / MySQL
- Real-Time Chat: Flask-SocketIO

## 🧩 Features
- Patient/Doctor login
- Real-time chat
- Health prediction
- Appointment scheduling
- Prescription uploads
- Video & wearable data

## 🚀 To Run the App

```bash
# Activate virtual environment
venv\Scripts\activate

# Install packages
pip install -r requirements.txt

# Run the app
python app.py
